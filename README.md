# 📄 V-Resume Builder Pro 🚀

A premium, modern, and 100% frontend-only resume builder designed to create high-scoring, ATS-friendly resumes in minutes. Built for everyone—from freshers and university students to seasoned corporate executives.

No backend, no server setup, and no logins required. Your data never leaves your browser.

## ✨ Key Functionalities

### 🧠 Smart ATS Engine & Optimization

* **Adaptive Layouts:** Toggle between **Fresher/Student** (Prioritizes Projects & Education) and **Experienced** (Prioritizes Work Experience) to instantly reorder your resume for maximum recruiter impact.
* **Live ATS Score Checker:** Analyzes your resume in real-time, giving you a score out of 100 with actionable, specific suggestions to improve your ATS compatibility.

### 🎨 Premium Design System

* **10 Professional Templates:** Hand-crafted, 100% ATS-readable layouts including:
* Modern Professional (Default)
* Fresher Resume
* University Student
* Internship Resume
* Experienced Professional
* Tech & Engineering
* Corporate Executive
* Minimalist Clean
* Creative & Design
* Elegant Premium


* **Dynamic Accent Colors:** Choose from 10 premium SaaS-style accent colors (Ocean Blue, Teal, Emerald, Crimson, etc.) that instantly update the UI and your resume template.
* **High-End Typography:** Select from 20 industry-standard and premium fonts (Inter, Poppins, Outfit, Plus Jakarta Sans, Playfair, Lora, etc.).

### ⚙️ Powerful Core Features

* **Real-Time Live Preview:** See your resume update perfectly on a realistic A4 digital paper as you type. Includes zoom controls (Zoom In, Zoom Out, Fit to Screen).
* **High-Quality PDF Export:** One-click download to a perfectly scaled, text-selectable PDF using `html2pdf.js`.
* **Dark Mode / Light Mode:** A beautifully designed UI that supports smooth toggling between deep slate dark mode and crisp light mode.
* **Mobile-Optimized Builder:** Fully responsive editor layout so you can update your resume seamlessly from a smartphone or tablet.

### 🔒 Secure Data Management

* **Auto-Save:** Everything you type is automatically saved to your browser's Local Storage. You will never lose your progress.
* **Data Sync (JSON Import/Export):** Download your entire resume profile as a `.json` backup file. Upload it later to restore your data instantly.
* **100% Privacy:** Because there is no backend or database, your personal information remains strictly on your device.

## 🛠️ Technology Stack

* **HTML5**
* **CSS3** (Custom Properties, Flexbox, CSS Grid, Glassmorphism aesthetics)
* **Vanilla JavaScript (ES6)** (Zero heavy frontend frameworks like React or Vue)
* **Libraries:** `html2pdf.js` (for PDF generation), FontAwesome (for icons)

## 🚀 How to Use / Run Locally

Because this project is strictly frontend, setting it up is incredibly simple:

1. Clone or download this repository.
2. Ensure `index.html`, `style.css`, and `script.js` are in the same folder.
3. Open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari).
4. Start building your resume!

## 🌐 Deployment

This application is completely static and can be deployed in seconds to free hosting services:

* **Vercel / Netlify:** Import your private GitHub repository and click deploy.
* **GitHub Pages:** Push to a public repository and enable GitHub Pages in the settings.
